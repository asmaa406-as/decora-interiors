# Decora Interiors - Interior Design Website

A modern, elegant website for an interior design studio specializing in warm and timeless home decor.

## 📋 Project Description

Decora Interiors is a complete static website for a professional interior design company. It features a clean, warm design that perfectly matches the home decoration field.

### Main Pages:
- **Home** – Hero section, About, Services, and FAQ
- **Projects** – Gallery with search functionality and "See More" button
- **Contact** – Interactive map, contact form, and team section


## 🚀 How to Run the Project

1. Download or clone the project folder.
2. Open the `HTML/home.html` file directly in your browser.
3. For better development experience, use **Live Server** (VS Code extension recommended).

**Note:** This is a static website and does not require a backend server.

## ✨ Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Consistent navigation bar and footer across all pages
- Warm, elegant color scheme suitable for interior design
- Interactive project gallery with **search** functionality
- "See More" button to load additional projects
- Interactive map using **Leaflet.js** on Contact page
- FAQ section with smooth open/close animation
- Clean and professional UI/UX

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Leaflet.js (for the map)
- Font Awesome (icons)

## 📌 How to Add New Projects

In `projects.html`:
- Add a new `.project-card` inside the `.card-grid`
- Use the `data-search` attribute to support the search feature
